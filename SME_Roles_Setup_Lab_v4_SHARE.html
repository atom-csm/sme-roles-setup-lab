Prompt 1 — สัมภาษณ์ (Interview)

วางในแชทธรรมดา (Default Sonnet, ไม่ต้องใน Project) → Claude ถามทีละข้อ → ตอบครบแล้วไป Prompt 2
ให้ Claude สัมภาษณ์คุณ เก็บข้อมูลทำ SKILL.md
Copy บล็อกด้านล่าง → วางในแชท → ตอบทีละข้อ (พิมพ์แค่ A/B/0/E) จนครบ 4 หมวด
⚠️
อย่าใจร้อน: ตอบทีละข้อให้ครบทั้ง 4 หมวดก่อน อย่าข้ามไป Prompt 2 จนกว่า Claude จะสรุป gap ("ต้องหาก่อน" / "รอได้") ให้ครบ
2

Prompt 2 — สร้าง SKILL.md

Universal Template · ใช้ต่อในแชทเดิมหลังตอบ Prompt 1 ครบ → ได้ไฟล์ SKILL.md กด Save Skill ได้ทันที
ให้ Claude เขียน SKILL.md จากบทสัมภาษณ์
ต่อในแชทเดียวกับ Prompt 1 — Claude จะสรุปทุกอย่างเป็นไฟล์ skill ที่ Save ได้
PROMPT 2 — Generate SKILL.md
💡
name ต้องเป็นตัวพิมพ์เล็กล้วน เป็นภาษาอังกฤษ ห้ามมี space/อักขระพิเศษ — แต่ละบทบาทใช้ name ของตัวเอง (เช่น ) เพื่อให้พิมพ์ เรียกได้ (ถ้าตั้งตัวใหญ่จะขึ้น "Unknown command")

Prompt 3 — ตั้ง Project

Universal Template · ใช้หลัง Save SKILL.md → ได้ Project Description + Instructions ไปวางใน Claude Project
ขั้นที่ 1 — สร้าง Claude Project
ตั้ง Project แยกไว้เป็นบ้านถาวรของบทบาทนี้ — แนบ Skill ที่ Save ไว้รอบที่แล้ว แล้วค่อยรัน Prompt สัมภาษณ์ในขั้นถัดไป
  1. เปิด Claude → กด + New Project (เมนูซ้าย) → ตั้งชื่อ
  2. ใส่คำอธิบายสั้น ๆ ในช่อง "What are you trying to achieve?"
  3. แนบ Skill / SKILL.md ที่ Save ไว้จาก Prompt 2 เข้า Project (Project knowledge / Files) เพื่อให้ Claude รู้จักธุรกิจตั้งแต่ต้น
  4. เปิดแชทแรกใน Project ที่เพิ่งสร้าง → พร้อมไปขั้นที่ 2
📌
ทำ Project ให้เสร็จก่อนวาง Prompt ข้างล่าง — Prompt สัมภาษณ์จะให้ผลดีที่สุดเมื่อรันอยู่ในแชทแรกของ Project ที่มี Skill แนบไว้แล้ว
ขั้นที่ 2 — สัมภาษณ์รอบ Project (วาง Prompt)
วาง Prompt นี้ในแชทแรกของ Project → ตอบ 4 หมวด → ได้ Project Description + Instructions ไปวางใน Project
PROMPT 3 — Project Setup Interview
ขั้นที่ 3 — วาง Description + Instructions ลงใน Project
เอา 2 อย่างที่ Claude ส่งมาในขั้นที่ 2 ไปตั้งค่าใน Project จริง — เสร็จแล้ว AI จะจำ context นี้ทุกแชทใน Project
  1. เปิด Project → กด ⚙️ Settings / Edit ของ Project
  2. วาง ① Project Description ลงในช่อง "What are you trying to achieve?"
  3. วาง ② Project Instructions ฉบับเต็มลงในช่อง Instructions ของ Project
  4. กด Save → เปิดแชทใหม่ใน Project เพื่อทดสอบว่า Claude เข้าใจ context
📌
Instructions อยู่ระดับ Project — จะมีผลกับทุกแชทใน Project นี้ ไม่ต้องวาง context ซ้ำทุกครั้ง ทำครั้งเดียวจบ

Prompt 6 — สร้าง Dashboard สรุปจากไฟล์ข้อมูล

วางไฟล์ข้อมูลในโฟลเดอร์ก่อน → ให้ Claude อ่านไฟล์แล้วสร้าง Dashboard สรุป + ชี้ปัญหา (ทำซ้ำได้ทุกเดือน)
ขั้นที่ 1 — ให้ Claude สร้างโครงสร้างโฟลเดอร์โปรเจค
วาง Prompt นี้ให้ Claude สร้างโฟลเดอร์ย่อยแบบเป็นระบบ (แยกข้อมูลดิบ / Dashboard / รายงาน / อ้างอิง) ก่อนเอาไฟล์ไปวาง — จะได้ทำงานเป็นระเบียบ หาของเจอ และทำซ้ำทุกเดือนได้
PROMPT 6.1 — สร้างโครงสร้างโฟลเดอร์
📁   ├─ 01_Data/   ├─ 02_Dashboard/   ├─ 03_Reports/   ├─ 04_Reference/   └─ 05_Archive/
📌
ทำขั้นนี้ครั้งเดียวตอนเริ่ม — เดือนถัด ๆ ไปแค่เอาไฟล์ใหม่ไปวางใน 01_Data/ แล้วรัน Dashboard ซ้ำได้เลย
ขั้นที่ 2 — วางไฟล์ข้อมูลลงในโฟลเดอร์
เอาไฟล์ข้อมูลที่มีอยู่ (Excel / CSV / export รายเดือน) ไปวางใน 01_Data/ ที่เพิ่งสร้าง แล้ว upload เข้า Project / แนบในแชท — Claude จะได้อ่านครบในที่เดียว
  1. เปิดโฟลเดอร์ 01_Data/ ที่ Claude สร้างให้ในขั้นที่ 1
  2. เอาไฟล์ข้อมูลทั้งหมดมาวางรวมกัน
  3. ตั้งชื่อไฟล์ให้สื่อ + มีเดือน เช่น เพื่อให้ Claude แยกช่วงเวลาได้ถูก
  4. เปิด Claude Project ที่ตั้งไว้ → ลากไฟล์ใน 01_Data/ เข้า Project Files (หรือแนบไฟล์ในแชท)
📌
วางไฟล์ให้ครบก่อนรัน Prompt ข้างล่าง — ถ้าไฟล์ไม่อยู่ใน Project/แชท Claude จะอ่านข้อมูลจริงไม่ได้ แล้วต้องเดา
ขั้นที่ 3 — ให้ Claude อ่านไฟล์แล้วสร้าง Dashboard สรุป
หลังวางไฟล์ครบแล้ว แทน [ระบุเดือน] ด้วยเดือนที่ต้องการ แล้ววาง Prompt นี้ใน Project ที่มีไฟล์อยู่
